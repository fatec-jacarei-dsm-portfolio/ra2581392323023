# Portfólio de Projetos - Fatec Jacareí

Portfólio acadêmico de **Vitor Cezar**, estudante de Desenvolvimento de Software Multiplataforma na Fatec Jacareí.

O projeto apresenta informações pessoais, projetos acadêmicos, atividades desenvolvidas durante a faculdade, cursos complementares e formas de contato.

## Acesso

O portfólio está disponível pela página publicada no GitHub Pages:

[Acessar portfólio](https://fatec-jacarei-dsm-portfolio.github.io/ra2581392323023/)

## Funcionalidades

- Alternância entre tema claro e tema escuro.
- Alternância de idioma entre português e inglês.
- Seção de projetos acadêmicos com links para os repositórios.
- Seção de atividades acadêmicas e cursos complementares.
- Layout responsivo para desktop e dispositivos móveis.
